== Description ==

Exclude IP’s From Google Analytics. You can paste google analytics code in this plugin and it will work.

Use your IP range to exclude Google Analytics for your IP’s

== Installation ==


1. Install and activate plugin.


2. Go to Settings>Analytics


3. Place your Google Analytics code


4. Place your IP's range

	- Example: You have IP’s from 111.111.111.1  to  111.111.111.10. 
	
	  You should include 111.111.111.1-10
